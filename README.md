# Manual interactivo DSpace · Sesión 1

Proyecto listo para publicar en Vercel.

## Qué contiene

- Manual interactivo sobre comunidades, subcomunidades, colecciones e ítems.
- Introducción a metadatos, formularios, esquemas y Dublin Core.
- Quiz interactivo.
- Checklist de revisión.
- Diseño responsive para computadora, tablet y celular.

## Publicación recomendada

1. Subir esta carpeta a un repositorio de GitHub.
2. Entrar a Vercel.
3. Importar el repositorio.
4. Dejar la configuración automática:
   - Framework: Vite
   - Build command: npm run build
   - Output directory: dist
5. Dar clic en Deploy.
